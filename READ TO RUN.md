### One-time setup (first time only)
1. Unzip this folder anywhere
2. Open a terminal or Command Prompt  in this folder
3. Make sure docker desktop or other equivilancey is running
4. Run this single command:
   docker-compose up --build
5. Got to http://localhost:8080
6. Create account or use test account (username: test password: test)
7. Voila! You can now shorten URLs
